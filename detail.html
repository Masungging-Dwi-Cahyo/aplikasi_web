Aplikasi web adalah program komputer yang dirancang untuk dijalankan di dalam browser web. Aplikasi web dapat diakses melalui internet dan tidak memerlukan instalasi di komputer pengguna. Aplikasi web terdiri dari berbagai komponen, termasuk HTML, CSS, dan JavaScript, yang dijalankan di dalam browser untuk memberikan pengalaman pengguna yang interaktif dan dinamis. Aplikasi web dapat berupa situs web sederhana seperti blog atau toko online, atau aplikasi web yang lebih kompleks seperti aplikasi perbankan online atau manajemen proyek. Keuntungan menggunakan aplikasi web adalah aksesibilitasnya yang mudah dan dapat diakses oleh pengguna dari berbagai jenis perangkat, seperti laptop, tablet, atau smartphone. Selain itu, aplikasi web juga mudah dikelola dan diupdate karena perubahan dapat dilakukan di server tanpa perlu melakukan instalasi di komputer pengguna. Namun, aplikasi web memiliki keterbatasan tergantung pada koneksi internet yang digunakan, dan tidak dapat digunakan ketika offline.
